# 🔍 Mini Search Engine (Full Stack)

## 🚀 Overview
A full-featured search engine built using:
- React (Frontend)
- Node.js + Express (Backend)
- Custom search engine core (DSA + System Design)

---

## ✨ Features
- 🔎 Full-text search
- 📊 TF-IDF ranking
- ⚡ Fast inverted index
- 🔤 Autocomplete (Trie)
- 🧠 Spell correction
- 🔍 Phrase search
- 🎯 Highlighted results
- ⚡ Query caching

---

## 🏗️ Tech Stack
- Frontend: React + Vite + TailwindCSS
- Backend: Node.js + Express
- Storage: JSON files

---

## 📁 Project Structure

frontend/ → UI
backend/ → APIs + search engine
docs/ → system design documentation
scripts/ → utilities


---

## ⚙️ Setup Instructions

### 1. Install dependencies

cd backend
npm install

cd ../frontend
npm install


---

### 2. Load data & build index

node ../scripts/loadSampleData.js


---

### 3. Run backend

cd backend
npm run dev


---

### 4. Run frontend

cd frontend
npm run dev


---

## 🌐 Open App

http://localhost:5173
